# Travaux Pratique pour .NET plate-form : gestion de processus

une application gérant deux classes de processus, les processus 
"ballon.exe" et "premier.exe".

![image](https://user-images.githubusercontent.com/15102793/33517693-0a597ed8-d789-11e7-929f-d8d59abeee09.png)


Voici le programmes pour le 1eme travail pratique dans la cadre de cours 
.NET plate-form. Le programme est en mode WPF qui peut lancer correctement
les deux applications, "ballon.exe" et "premier.exe". Ces deux processus 
"ballon.exe" et "premier.exe" seront dans le répertoire courant,

## Comment utiliser ?

Importez le projet 

1. File -> Open -> Project/Solution -> choisissiez `MultiProcessExample.sln`
2. Lance le programme par cliquer le bouton `Start`

Ou allez dans le répertoire `MultiProcessExample\WpfApp1\bin\Debug` et 
puis lancez `WpfApp1.exe`
